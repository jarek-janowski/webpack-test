Co to jest CSS

Język służący do opisu formy prezentacji (wyświetlania) stron WWW.

Trochę historii

Początki

CSS został opracowany przez organizację W3C w 1996 r. jako potomek języka DSSSL przeznaczony do używania w połączeniu z SGML-em. Pierwszy szkic CSS zaproponował w 1994 r. Håkon Wium Lie

Pierwsza specyfikacja

Pod koniec 1996 roku wydano oficjalną dokumentację CSS, Kaskadowe arkusze stylów, poziom 1.
